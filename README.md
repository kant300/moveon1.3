## 💡 프로젝트 개요
**무브온(mOveOn)** 은 새로운 지역으로 이사하거나 발령받은 사람들이  
필요한 생활·공공·안전 정보를 한눈에 확인하고 정착 스트레스를 줄일 수 있도록 돕는 **공공데이터 기반 앱**입니다.  

> “낯선 동네에서도 따뜻하게 정착할 수 있도록” 🌱


## 🎯 프로젝트 목표
- 💬 지역 정착에 필요한 **정보 파편화 문제 해결**
- 🛡️ **성범죄자 지도**와 **실시간 CCTV 지도**로 체감 안전 강화
- 🗺️ 공공데이터를 활용한 **통합 생활 정보 제공**
- 🤝 지역 커뮤니티 연결을 통한 **사회적 고립 완화**


## 🧩 주요 기능
| 카테고리 | 기능 내용 |
|-----------|------------|
| 🏠 생활정보 | 관공서 위치, 심야약국/병원, 의류수거함, 쓰레기배출, 폐가전수거, 전입신고 , 공과금정산(추후예정) |
| 🛡️ 안전정보 | 성범죄자 지도, 실시간 CCTV 지도, 비상급수시설, 대피소, 공중화장실, 민간구급차 |
| 🚌 교통정보 | 지하철(배차정보/승강기 고장여부), 버스정류장, 전동휠체어충전소, 공용주차장 |
| ❤️ 커뮤니티 | 소분모임, 지역행사, 중고장터, 동네후기, 구인/구직 |

## 🧠 기획 배경
현대인의 주거 이동은 잦지만,  
정착에 필요한 정보는 **각 기관·앱·포털에 흩어져 있습니다.**  

공공데이터를 하나로 엮어  
‘**새로 이사한 사람의 첫 3일/3주/3개월을 돕는 서비스**’로  
무브온을 기획했습니다.
